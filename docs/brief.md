## Sản phẩm cần build
REST API backend cho nền tảng học tiếng Anh. Xử lý auth, quản lý lớp/bài tập, OCR nhận dạng nội dung, chấm điểm speaking bằng Whisper AI, gamification.

## Must-have features
- Auth (JWT) + RBAC (Admin / Teacher / Student)
- CRUD: Users, Classes, Exercises, Vocabulary, Scores
- OCR: nhận dạng text từ ảnh/PDF
- Speaking: nhận audio, chấm điểm bằng Whisper AI
- Gamification: điểm, sao, huy hiệu, streak, leaderboard
- Admin: quản lý user, phân quyền, audit log
